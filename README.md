# Shopping Mall Sales Dashboard

## Project Overview
This Excel dashboard provides insights into shopping mall sales performance by analyzing revenue, customer purchases, product categories, and sales trends.

## Objectives
- Track total revenue generated.
- Monitor total quantity of products sold.
- Analyze customer purchasing patterns.
- Identify top-performing product categories.
- Support data-driven business decisions.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

## Key Metrics
- Total Revenue
- Total Customers
- Total Quantity Ordered
- Category Performance
- Sales Distribution

## Dashboard Features
- Interactive filters and slicers
- Revenue analysis
- Product category breakdown
- Customer purchase analysis
- Visual KPI tracking

## Key Insights
- Revenue is concentrated in a few high-performing categories.
- Certain customer segments contribute significantly to sales.
- Product demand varies across categories.
- Sales trends reveal opportunities for inventory optimization.

## Recommendations
- Increase inventory for high-performing products.
- Develop targeted promotions for key customer segments.
- Monitor underperforming categories and adjust strategies.
- Use customer insights to improve marketing campaigns.
